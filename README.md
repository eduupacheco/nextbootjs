# NextBootJS

Now you can build your NextJS and Bootstrap 5.3 skeleton app faster with NextBootJS.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
